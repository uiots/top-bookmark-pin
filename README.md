# Top Bookmark Pin

点击扩展图标，将当前页面收藏到**书签栏最开头**（而非 Chrome 默认的末尾）。

## 动机

Chrome 原生的书签收藏操作默认将新书签添加到书签栏末尾。对于高频使用的收藏场景，每次都滚动到最底部才能找到刚收藏的页面体验不佳。本插件解决了这个问题：一键收藏、自动置顶。

## 用法

1. 安装扩展（开发者模式加载 `src` 目录）
2. 浏览任意网页时，点击扩展图标
3. 页面将被收藏到书签栏第一个位置
4. 再次点击同一页面可取消收藏（从书签栏移除）

## 实现

- Manifest V3
- 使用 `chrome.bookmarks.create()` 的 `index: 0` 参数实现置顶
- 点击切换：已存在则删除，不存在则添加
- 兼容中英文版 Chrome 的书签栏文件夹名称
